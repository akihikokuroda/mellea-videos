The `/api/users` endpoint retrieves a list of all users. Each user object contains the following fields:

### Example Code

Here is a simple example of how to call this endpoint using Python and the `requests` library:

```python
import requests

url = "http://example.com/api/users"
response = requests.get(url)

    users_data = response.json()
    for user in users_data['users']:
        print(f"ID: {user['id']}, Username: {user['username']}, Email: {user['email']}")

    print("Failed to retrieve data")
```
